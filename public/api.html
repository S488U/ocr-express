API Documentation

POST /ocr
Multipart/Form-Data

Accepts image files or URLs to process via the OCR engine.

Parameters

  • images One or more image files (binary).
  • urls JSON Array of URL strings (optional).

Example Request

curl -X POST https://ocr.plexaur.com/ocr \
  -F "images=@screenshot.png" \
  -F "images=@document.jpg"

Example Response

{
  "results": [
    {
      "name": "screenshot.png",
      "type": "file",
      "text": ["Extracted text line 1", "Line 2"],
      "time": 0.45
    }
  ],
  "totalTime": 0.45
}
